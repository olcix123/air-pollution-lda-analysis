# air-pollution-lda-analysis

This repository contains my individual part of a team project focused on Linear Discriminant Analysis of air pollution data.

## Project scope
- data preparation for LDA,
- definition of explanatory and grouping variables,
- calculation of discriminant functions,
- classification matrix analysis,
- leave-one-out cross-validation,
- interpretation of results.

## Tools
R, RStudio

## Result
The LDA model achieved 75% overall classification accuracy. The year 2014 was the easiest group to distinguish, while 2015 was the most difficult to classify.
